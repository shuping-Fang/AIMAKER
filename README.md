# AIMAKER
2025 KISS SCIENCE 活動由台灣半導體研究中心籌辦，超維顧問工作室承辦，以「AI創客 - 挑戰 DIY 智慧家庭語音系統」為主題，讓參觀者從體驗式互動中學習人工智慧與語音辨識的核心原理。場域將結合創客實作、AI模型訓練與物聯網應用，展示聲音如何成為控制智慧裝置的介面。透過模擬日常語音指令（如開燈、關燈等），學員可在現場實作從語音資料收集、模型訓練、部署至ESP32微控制器的完整流程。現場並設有實體裝置展示區，呈現由語音控制的LED燈、風扇與門鎖系統，呈現語音辨識技術與生活應用的整合成果。

本場次為動手做＋互動體驗，總時長150分鐘，包含語音技術基礎、AI 模型訓練、裝置控制與成果發表四大主軸。課程內容如下：

1. 語音與AI技術導入簡介：介紹聲音的數位特徵（如MFCC）、語音辨識應用場景，並示範語音控制生活裝置的範例，提升學員學習動機。
2. 語音資料現場收集：分組錄製如「開燈」「關燈」等指令，建立個人化語音樣本資料集，理解資料多樣性對模型準確度的影響。
3. Edge Impulse平台語音分類模型訓練：操作雲端平台進行MFCC特徵提取、神經網路訓練與即時測試，體驗從資料到模型的過程。
4. 模型部署與Arduino編程教學：將訓練完成的模型匯出至Arduino IDE，編寫ESP32語音控制程式，對應控制LED燈、風扇與電磁門鎖。
5. 語音控制互動體驗：參與者親自對ESP32裝置說出語音指令，觀察模型辨識反應與裝置啟動情形，並協助他人測試與修正指令反應。
6. 展示成果分享：每組簡要介紹其訓練流程與控制成果，回顧整體學習重點並與他組作品互動交流。
